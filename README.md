# Code_Html

本项目用于将编程语言代码源文件转换为 html 网页文件。

Html文件会根据代码的句法（比如关键字）将其以不同的颜色或字体显示，
以一种更加友好的方式显示代码。

This project is aim at conveting programming Language 
source files such as C\C++(.c.h),java(.java) into html files.

The html files, which can show the codes with different color,font according to it's syntax
(e.g reserved word/key word), will provide a ui-friendly view to user.
